# Model Presets

Recommended model and thinking-level combinations for different use cases.

## Usage

These are reference configurations. Apply them via the Mux settings UI or by editing `~/.mux/config.json` under `agentAiDefaults` or `subagentAiDefaults`.

## Available Presets

| File | Use Case | Model | Thinking |
|------|----------|-------|----------|
| `fast.json` | Quick iteration, simple tasks | Claude Sonnet | Low |
| `deep.json` | Complex planning, architecture | Claude Opus | High |
| `balanced.json` | General development | Claude Sonnet | Medium |
